Promedio 1

Aqui se irá subiendo todos los avances de
las clases virtuales y presenciales.

Semana 1

En la primera semana se realizo la presentación del curso, las pautas y rúbricas de calificación. Además que se prensento ejemplos de proyectos innovadores y todo lo que debe tener para que este sea funcional y en la clase presencial realizamos un ejercicio.

Semana 2

En la segunda semana aprendimos a usar la aplicación de Visual Code para crear una página web. Aprendimos a crear un archico index.html, a subir imagenes, colocar links y subir imagenes. En la clase presencial se realizo un avance del proyecto grupal.

Semana 3

Se comenzó a usar estilos mediante el css para poder darle color, una fuente a cada espacio, darle posición y efecto a los objetos que se pongan y en la clase presencial se hizo un repaso y también se avanzo con la idea del proyecto grupal.

Semana 4

Seguimos viendo los estilos y las estructuras de los códigos de añadir vinculos para poder navegar en otras páginas, aprender a centrar y diagramar como queremos que sea nuestra página web. En clase prensencial también se realizo una exposición del avance del proyecto grupal.

Semana 5

Se presentará las bitacoras de manera individual y en presencial se realizará la exposición de los proyectos grupales.
